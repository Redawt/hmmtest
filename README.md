# HMM
A Toy implementation of Hidden Markov Model (HMM)

## Intro
This implementation has following algorithms for HMM.
- Forward/Backward (sum-product)
- Viterbi (max-product)
- Baum-Welch (EM with exact E)
- Gibbs (EM with approx E)
